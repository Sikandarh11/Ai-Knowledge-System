Node.js v18 or above — download from nodejs.org
download and install Node.js, which includes npm (Node Package Manager).



# 1. Clone or receive your project folder
# then navigate to frontend
cd Agentic-AI-Personal-Knowledge-System/frontend

# 2. Install all dependencies (reads package.json automatically)
npm install

# 3. Create the .env file manually in frontend/ and add the following line to it (local dev):
VITE_API_BASE_URL=https://backend-ai-knowledge-system-9jf3.vercel.app/docs

# For Vercel production, set VITE_API_BASE_URL in Vercel Environment Variables
# to your deployed backend URL (example: https://your-backend-project.vercel.app)

# 4. Run the project
npm run dev

# 5. Open browser at
http://localhost:5173
